DOM Manipulation

    Two Step Process
  1. Select element node
  2. Use a method or property to work with that element's attributes

Selecting Individual and Multiple Elements

Method Description
geElementbyId() uses the value of an element's id attribute (which should be unique within the page)
querySelector() uses CSS selector, and returns the first matching element
setAttribute() sets the value of an attribute
removeAttribute() removes an attribute from an element node

once you have an element node you can use methods on that element node to access and change its attributes

Attribute Nodes

Method Description
getAttribute() get the value of an attribute
hasAttribute() checks if element node has a specified attribute
setAttribute() sets the value of an attribute
removeAttribute() removes an attribute from an element node

Manipulating a DOM Node's Styles

You can access and change the styles of a DOM nodes via the style property

        
          CSS:
          body {
            color: red;
            background-color: pink;
            padding-top: 10px;
          }

          //Applying the Same Styles via JavaScript:

          var pageNode = document.body;
          pageNode.style.color = 'red';
          pageNode.style.backgroundColor = 'pink';
          pageNode.style.paddingTop = '10px';
