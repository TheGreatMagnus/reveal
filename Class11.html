Array methods

  1. mutator methods: modify the array. examples: pop(); shift(); push(); splice() (and others!)

  2. accessor methods: return a new value or representation. concat(); join(); slice()

  3. iteration methods: executes a provided function once for each array element.

Iteration methods

iteration methods: methods that operate on every item in an array, one at a time. These methods are closely associated with loops.


  • forEach()
  • map()

Looping through an Array

						
							//using a for loop
							var donuts = ["jelly donut", "chocolate donut", "glazed donut"];

							for (var i = 0; i < donuts.length; i++) {
							  donuts[i] += " hole";
							  donuts[i] = donuts[i].toUpperCase();
							  console.log(donuts[i]);
							}
						
					
						
							//using forEach() method
							var donuts = ["jelly donut", "chocolate donut", "glazed donut"];

							donuts.forEach(function(donut) {
							  donut += " hole";
							  donut = donut.toUpperCase();
							  console.log(donut);
							});

forEach() syntax

					 
					var fish = [ "piranha", "barracuda", "cod", "eel" ];

					fish.forEach(function (individualFish){
    				console.log(individualFish);
					})

					//output
					piranha
					barracuda
					cod
					eel

forEach() parameters

The function that you pass to the forEach() method can take up to three parameters.

  1. element: The value of the current element being processed in the array
  2. index: The index of the current element being processed in the array. (optional)
  3. array: The array that forEach() is being applied to. (optional)

map()

creates a new array with the results of a function call on each element in the array.

					 
							var donuts = ["jelly donut", "chocolate donut", "glazed donut"];

							var improvedDonuts = donuts.map(function(donut) {
							  donut += " hole";
							  donut = donut.toUpperCase();
							  return donut;
							});

							//improvedDonuts array: ["JELLY DONUT HOLE", "CHOCOLATE DONUT HOLE", "GLAZED DONUT HOLE"]
