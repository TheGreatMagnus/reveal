Arrays!

An array is a collection of items that can be modified and looped over.

Review

  1. Create an array: array literal notation []
  2. Access values: 0 indexed
  3. Add values: push and unshift
  4. Remove values: pop and shift
  5. Merge values: concat
  1. mutator methods, modify: pop(); shift(); push(); splice() (and others!)
  2. accessor methods, return a new value or representation: concat(); join(); slice()
  3. iteration methods: methods that operate on every item in an array, one at a time. These methods are closely associated with loops.

but one of the greatest love stories of all time is that between a data structure known as an array and the for loop. The bulk of your time, you will be using loops to travel through all of the data stored in an array, so just glance through the following code and explanation.

Looping through an Array

						
							var donuts = ["jelly donut", "chocolate donut", "glazed donut"];

							for (var i = 0; i < donuts.length; i++) {
							  donuts[i] += " hole";
							  donuts[i] = donuts[i].toUpperCase();
							  console.log(donuts[i]);
							}
						
					
						
							var donuts = ["jelly donut", "chocolate donut", "glazed donut"];

							donuts.forEach(function(donut) {
							  donut += " hole";
							  donut = donut.toUpperCase();
							  console.log(donut);
							});

method executes a provided function once for each array element

forEach() parameters

The function that you pass to the forEach() method can take up to three parameters.

  1. element: The value of the current element being processed in the array
  2. index: The index of the current element being processed in the array. (optional)
  3. array: The array that forEach() is being applied to. (optional)
