Functions

At a very basic level, a function is nothing more than a wrapper for some code. Functions...

  1. Groups statments together
  2. Makes your code reusable

Parts of a function

  • function keyword
  • name of the function (use camelCase) followed by ()
  • open and closing brackets that will enclose any statements {}
  • function call

Two Steps to working with functions

1.Declaring a function: creating a function

2.Calling a function: executing all the statements within a function with just one line of code.

Declaring a Function

Calling a function

Write the following bit of code between script tags in your html document.

      
          function sayHello() {
            document.write('Hello!');
          }
          sayHello();

*note* console.log is for printing things to the console. document.write is for printing things to an html document.

Rewriting our code as functions

      
        function myName() {
          var firstName = "Ramiro";
          var lastName = "Rodriguez";
          document.write("My name is " + firstName + " " + lastName + ".");
        }
        myName();

Another example

      
        function showDistance() {
          var speed = 55;
          var time = 4;
          document.write("My distance is " + speed*time + ".");
        }
        showDistance();

Functions with Parameters

Parameters: noun. Information. Stuff.

Parameters

1. the information the function needs to perform its task

2. parameter values are called arguments

3. arguments can be values (i.e. specific numbers) or variables

Declaring Functions that need Information

You specify your arguments to the function as part of the function call:

        
          function showDistance(speed, time) {
            alert(speed * time);
          }
          showDistance(10, 5);

When the showDistance function gets called, the 10 corresponds to the speed argument, and the 5 corresponds to the distance argument. That mapping is entirely based on order.

The Return Keyword

used for returning data from a function

      
        function getDistance(speed, time) {
            var distance = speed * time;
            return distance;
        }
