Switch statement

A switch statement tests a value and can have many case statements which defines various possible values. Statements are executed from the first matched case value until a break is encountered.

Switch statement syntax

							
								swtch(some expression) {
									case 'option 1':
										//Do something;
										break;
									case 'option 2':
										//Do something else;
										break;
									case 'option 3':
										//Do yet another thing;
										break;
								}

Switch statement example

								
									function caseInSwitch(val) {
									  var answer = "";
									  switch(val) {
									    case 1:
									      answer = "alpha";
									      break;
									    case 2:
									      answer = "beta";
									      break;
									    default:
									      answer = "something else";
									  }
										  return answer;
										}
										caseInSwitch(3);

Loops:

generic solution for repeating code with control over how many times the code repeats

There are 3 kinds of loops you can create:

  1. while loops
  2. for loops
  3. do...while loops

Intro to Loops

While Loops: if you donâ€™t know how many times the code should run, you can use a while loop.

Loops check a condition. If it returns true, a code block will run. Then the condition will be checked again and if it still returns true, the code block will run again. It repeats until the condition returns false.

Parts of a While Loop

when to start, when to stop, and how to get to the next item
