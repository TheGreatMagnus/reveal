Creating an Object

Object in JavaScript is just key-value pairs stored in a curly braces (a hash).

Property or Method

  • a property is a value stored in the hash key
  • a method is a function stored in hash key

Property or Method?

var umbrella = { color: "pink", isOpen: true, open: function() { if (umbrella.isOpen === true) { return "The umbrella is already opened!"; } else { umbrella.isOpen = true; return "Julia opens the umbrella!"; } }, };
