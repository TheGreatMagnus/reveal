For Loop Syntax

			for (var i = 1; i <= 50; i++) {
			sum = sum + i;
			}
			alert("Sum = " + sum);    // => Sum = 1275

3 Parts of a For Loop


  1. initializer (var i = 1;)
  2. test condition (i <= 50;)
  3. updater (i++, or i--)

In a for-loop, all three parts i.e. initializer, test condition, and updater are written together in a single line (called an iteration statement).

When to Use one loop vs another

In general, you should use a for loop when you know how many times the loop should run. If you want the loop to break based on a condition other than the number of times it runs, you should use a while loop.
