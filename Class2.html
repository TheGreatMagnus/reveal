Modulus (Remainder) Operator %

The modulus operator returns the modulus left over when one operand is divided by a second operand.

							
								-1 % 2 // -1
								1 % 2 // 1
								2 % 3 // 2
								-4 % 2 // -0
								5.5 % 2 // 1.5
								12 % 5 // 2

Check if a number is divisible by another number. If the remainder is 0, it is. 

							
								console.log(456028 % 659)
