Remainder Operator %

The remainder operator returns the remainder left over when one operand is divided by a second operand.

							
								12 % 5 // 2
								-1 % 2 // -1
								1 % 2 // 1
								2 % 3 // 2
								-4 % 2 // -0
								5.5 % 2 // 1.5
