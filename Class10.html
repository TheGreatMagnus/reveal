Add values

push: adds an element to the end of an arrray

unshift: adds an element to the beginning of an array

						
							var groceries = ["milk", "bread", "cereal"];
							groceries.push("bananas");
							groceries.unshift("spaghetti");
							console.log(groceries);
							// [ 'spaghetti', 'milk', 'bread', 'cereal', 'bananas' ]

*note* Both the push and unshift methods, besides adding the elements to the array when you use them, return the new length

Remove values

pop: removes the last element from the array

shift: removes the first element from the array

					
						var groceries = ["milk", "bread", "cereal"];
						groceries.pop();
						groceries.shift();
						console.log(groceries);
						// [ bread' ]

For a list of array methods, https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array

Merge values

concat: creates a new array that is made up of two separate arrays. 

					
						var good = ["Mario", "Luigi", "Kirby", "Yoshi"];
						var bad = ["Bowser", "Koopa Troopa", "Goomba"];

						var goodAndBad = good.concat(bad);
						//[ 'Mario',
							  'Luigi',
							  'Kirby',
							  'Yoshi',
							  'Bowser',
							  'Koopa Troopa',
							  'Goomba' ]

Concat returns a new array that stores the results of our concatenation operation. The order of the elements inside goodAndBad is good first and bad second
